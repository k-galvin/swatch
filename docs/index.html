The Story

Swatch was born from the need for a simple, declarative way to describe architectural spaces. While professional CAD software is powerful, it is often too heavy for quick ideation and algorithmic layout generation. Swatch bridges the gap between manual drafting and automated design.

We believe that architectural logic should be human-readable and programmable. By providing a lightweight syntax for layouts, walls, and furniture, Swatch allows designers to focus on spatial relationships while leveraging the power of loops and components to create complex, generative floor plans.

Every Swatch script compiles directly into clean, scalable SVG, making it easy to share, print, and integrate into modern web workflows. Whether you are drafting a simple kitchen or generating a massive library, Swatch provides the primitives you need to bring your vision to life.

Examples

Modern Studio

component ModularSofa(x, y, pieces) {
  for p in 0 ..< pieces {
    place Section at [x + p * 50, y];
  }
}

Layout "Studio" size [800, 600] {
  Wall North from [50, 50] to [750, 50];
  ModularSofa(300, 400, 3);
}

Generative Row

Layout "Generative" size [500, 200] {
  repeat 5 {
    place Chair at [x, 100];
    x = x + 80;
  }
}

Kitchen Logic

Layout "Kitchen" size [400, 400] {
  place Counter at [200, 50];
  if hasIsland {
    place Island at [200, 200];
  }
}

Office Desk

Layout "Office" size [300, 300] {
  place Desk at [150, 150] [color: #34495e];
  place Chair at [150, 200] [rotate: 180];
}

Art Gallery

Layout "Gallery" size [600, 400] {
  Wall W from [100, 100] to [500, 100];
  place Art at [300, 100] [label: "Mona Lisa"];
}
